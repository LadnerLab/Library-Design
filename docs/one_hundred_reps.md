# One Hundred Reps

## Overview

Optional Module! Remove identical protein sequences, including those that are a subset of another target [onehundredreps.py]

### Tutorial/Use

Command:
```
onehundredreps.py \
  -f poxviridae_unaligned_30AA.fasta \
  -r poxviridae_unaligned_30AA_100rep.fasta \
  -m 100_rep_map.txt
```
